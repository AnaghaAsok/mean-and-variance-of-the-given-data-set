To run this program 

1. Using .jar file

1.1 Start Hadoop
1.2 add the file numbers.txt to the local host.
1.3 type hadoop jar q3.jar /<path>/numbers.txt /<path>/output_3

PS: I have created a runnable .jar file so classpath need not to be given explicitly.

2. Using Code
1.1 Build the code in eclipse.
1.2 create runnable .jar file by exporting and launching right configuration
1.3 Reapeat above steps.
